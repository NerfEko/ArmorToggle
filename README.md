# ArmorToggle

ArmorToggle by EvilEKo. A simple Minecraft Fabric mod that allows you to toggle the visibility of armor with a keybind. Perfect for showing off your skin while keeping your armor equipped!

## Features
- Toggle armor visibility with a single key press.
- Configurable keybind (default: **G**).
- Client-side only.

## Installation
1. Install [Fabric Loader](https://fabricmc.net/).
2. Download the latest version of this mod and place it in your `mods` folder.
3. (Optional) Install [Fabric API](https://curseforge.com/minecraft/mc-mods/fabric-api) for better compatibility.

## Usage
Press **G** (default) to toggle armor visibility. You can change this keybind in the Minecraft controls settings.
